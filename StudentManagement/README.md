# Student Management System

## Features
- ASP.NET Core Web API
- JWT Authentication
- CRUD Operations
- SQL Server
- Swagger
- Serilog Logging
- Exception Middleware

## Setup
1. Open project in Visual Studio
2. Update SQL Server connection string
3. Run migrations
4. Start project
5. Open Swagger

## API Endpoints
- GET /api/students
- POST /api/students
- PUT /api/students/{id}
- DELETE /api/students/{id}
